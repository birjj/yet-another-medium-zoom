Yet Another Medium Zoom

A library by birjolaxew

YAMZ is a lightbox library heavily inspired by medium-zoom (which is in turn inspired by Medium).

It provides an easy way to add smooth, fully featured and highly customizable Medium-like zooming to your images.

It supports captions, high-res images, albums, and so much more.

Should you wish to have more control over how the lightbox is displayed, you can style it with CSS or completely replace the DOM structure - the animation will adapt automatically.
